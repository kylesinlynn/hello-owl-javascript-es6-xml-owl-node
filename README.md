## Prerequisites
- `Nodejs`
- `NPM or alternative`

## Project Setup
1. Clone the git repository
    ```bash
    $ git clone git@github.com:kylesinlynn/hello-owl-javascript-es6-xml-owl-node.git
    ```

2. Move directory and install the require modules
    ```bash
    $ pnpm i
    ```

3. Run in development mode
    ```bash
    $ pnpm run dev
    ```
